Based on the argument provided to the will_script.py a system command is return from the cmd.py and then executed. We should notice the whitelist array of command we are allowed to use.

I personally tried to insert a simple import os; os.system("/bin/bash")
command at the top of the cmd.py with sed and then with awk but I was not successful, so I used SSH to get a proper shell and then I used nano to edit the script and insert this line on the top of the file.

How to SSH into the machine?

  1. Create key pairs for SSH with ssh-keygen command.
  2. Copy the id_rsa.pub file to the authorized_keys file with cp id_rsa.pub authorized_keys command inside the .ssh directory.
  3. Copy the content of the private key (id_rsa) to your local machine.
  4. Set permissions of the private key with chmod 600 id_rsa command.
  5. Login as mat user with ssh -i id_rsa mat@IP
